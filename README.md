# ❌⭕ Tic Tac Toe Game

A classic Tic Tac Toe game built using HTML, CSS, and JavaScript. Play with a friend or challenge yourself in this simple and fun game!

## 🎮 Features
- Two-player mode
- Interactive game board
- Win / Draw detection
- Game reset option

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 How to Play
1. Player X starts the game
2. Players take turns marking X and O
3. First player to align 3 marks (row, column, or diagonal) wins
4. If all cells are filled and no winner → it's a draw
5. Click **Reset** to play again
